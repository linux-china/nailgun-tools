Nailgun Tools
=============

Nailgun tools is a Nailgun scripts to include some features such as groovy integration.

### Groovy Integration

    $ ng-groovy-server
    $ ng-groovy hello.groovy

### How to install

    $jenv selfupdate
    $jenv install nailgun
How to install jenv? Please reference https://github.com/linux-china/jenv
